# Evolve HQ: Progress Summary & Roadmap

## 📊 Project Overview
Evolve HQ is a premium, production-grade workspace, admin console, and CRM platform.

## ✅ Completed Modules

### 1. Core Infrastructure
- **Framework**: Next.js 16+ with MongoDB/Mongoose.
- **Authentication**: NextAuth v5 with Role-Based Access Control (Admin, Superadmin, Employee).
- **Timezone System**: Standardized IST-safe date handling across the app.
- **Real-time**: Pusher integrated for live updates.

### 2. HQ Workspace (Employee Portal)
- **Bento Dashboard**: High-end visual layout for employees.
- **Attendance Management**:
  - Live clock-in/out with session timer.
  - Work mode selection (Office/WFH).
  - Attendance history and detailed logs.
- **Personal Stats**: Real-time tracking of tasks, streaks, and leaves.
- **Real-time Chat**: Channel-based messaging with persistence.

### 3. Admin Console (HR & Operations)
- **Dashboard**: Unified metrics (headcount, tasks, leaves).
- **Team Presence**: Live tracking of who is clocked in, on leave, or active.
- **Leave Management**: Full approval workflow for leave requests.

### 4. CMS Editor (Content Management)
- **Unified Interface**: Glassmorphism dashboard for managing external site content.
- **Projects Tab**: Full CRUD for portfolio items, gallery management, and case studies.
- **Services Tab**: CRUD and reordering for agency services.
- **Testimonials Tab**: Management of client feedback.
- **FAQ Tab**: Management of frequently asked questions.
- **Inquiries Tab**: View and manage leads from the public site.
- **Lead Pipeline**: Kanban-style interface for tracking sales leads with drag-and-drop.
- **Lead Details**: Integrated dialogs with activity logs and history.
